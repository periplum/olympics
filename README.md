# olympics — every Olympic host city

[![Built with Periplum](https://img.shields.io/badge/built_with-Periplum-4da3ff)](https://periplum.js.org)

Every **Summer** and **Winter** Olympic Games host city, 1896–2032, in chronological order
— ☀️ for the Summer Games, ❄️ for the Winter Games.

**[▶ Open the map](https://periplum.github.io/olympics/)** — press play to watch the Games
travel the globe, alternating sun and snow, or drag the date slider.

This POC shows off Periplum's **`statusIcons`** feature (emoji markers by status). The host
list is curated in [`build.py`](build.py); re-run it when a new host is awarded (≤ once a
year). A showcase consumer of **[Periplum](https://github.com/periplum/periplum)**.
